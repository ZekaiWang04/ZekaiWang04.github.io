Roboprompt (ICRA 2025)

Historically, for a robot to learn a task, we need to collect task-specific data and train a model from scratch. With the advances in Large Language Model, Vision Language Model, and robotics foundation models, we can now efficiently finetune a foundation model to perform a task. However, the process of collecting data and finetuning is still required. In this work, we present Roboprompt, a framework that utilizes pretrained LLM directly to perform a task without any finetuning. We are inspired by recent works investigating LLM's in-context learning abilities and propose a method to generate in-context examples for a specific task. With these generated examples, we show a LLM can successfully perform the task without any finetuning. For more details, visit the project website and read the paper on arXiv.

CS 180 Computer Vision and Computational Photography Projects

Below are some of the projects I worked on during the CS 180 course.

Project 1: Image Stitching

In this project, we examined different similarity scores to align the RGB channels of an image. We also used edges as features to help align the images. For more details, visit the project website.



Project 2: Filter and Frequencies

We implemented hybrid image blending (blend two images such that when looked upfront the hybrid image looks like one original image and when looked from a distance it looks like the other original image) and pyramid blending (for a smooth transition region). For more details, visit the project website.



Project 3: Face Morphing

Given the images of two faces and their corresponding keypoints (e.g. from a Neural Network), we morph the two faces together by generating a triangle mesh, calculate the mean shape of the keypoints, and morph the corresponding triangles together using linear algebra. For more details, visit the project website.



Project 4: Panorama Stitching

Given two images that shares a common projection center, we can use perspective transform to stitch the two images together. We implemented the full pipeline of corner detection, Adaptive Non-Maximal Suppression, corner feature descriptors, feature matching, and 4-point RANSAC. For more details, visit the project website.



Project 5: Diffusion Models

We played around with a trained diffusion model and implemented Classifier-Free Guidance for sampling, as well as its applications (e.g. SDEdit). In addition, we trained a simple diffusion model on MNIST to generate hand-written digits. For more details, visit the project website.



Project 6: Neural Radiance Fields (NeRF)

We implemented Neural Radiance Fields, a method trained on different views of a scene and is capable of rendering from novel views. For more details, visit the project website.
